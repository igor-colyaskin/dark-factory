# .roo/rules/global.md — Global Instructions

## Language
- Communication with user: Russian
- Code comments: English
- Documentation files: English
- Variable/function names: English
- Git commits: English

## Code Style
- JavaScript: ES modules (import/export), no var, const by default
- Semicolons: yes
- Quotes: single
- Indent: 2 spaces
- No console.log in production code (use a logger or remove)
- Meaningful variable names, no abbreviations except common ones (req, res, err)

## Files
- UTF-8 everywhere
- LF line endings
- Newline at end of file
- No trailing whitespace

## AI Behavior
- Do not modify CONCEPT.md, TRACKER.md, BOOTSTRAP.md
- Do not commit to git without explicit approval
- When unsure, ask — do not guess
- Respond concisely, no unnecessary explanations unless asked
